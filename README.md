# Приложение для автоматизации лабораторного учета сырья

## Описание

Приложение с графическим интерфейсом для учета поступления сырья, генерации актов и ведения журнала в Excel.

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка путей

Пути задаются в файле `settings.json` (лежит рядом с программой).  
При переносе на другой компьютер обычно достаточно заменить значения там.

Ключевые параметры:

- `excel_file` — путь к Excel-журналу (таблица)
- `acts_folder` — папка, куда сохраняются акты
- `word_template` — путь к шаблону Word
- `database_file` — файл базы данных SQLite

## Подготовка файлов

1. **Excel-файл**: Создайте файл `пример таблицы.xlsx` с заголовками столбцов:
   - № акта
   - Наименование
   - Поставщик
   - Производитель
   - № партии
   - Вес
   - Дата поступления
   - № вагона/авто
   - Дата отбора
   - Вид упаковки
   - Физическое состояние
   - Статус
   - Примечание

2. **Шаблон Word**: Создайте файл `шаблон.docx` с placeholder'ами:
   - `{{ACT_NUMBER}}` - номер акта
   - `{{DATE}}` - текущая дата
   - `{{TIME}}` - время проверки
   - `{{NAME}}` - наименование
   - `{{SUPPLIER}}` - поставщик
   - `{{MANUFACTURER}}` - производитель
   - `{{BATCH}}` - № партии
   - `{{WEIGHT}}` - вес
   - `{{ARRIVAL_DATE}}` - дата поступления
   - `{{VEHICLE}}` - № вагона/авто
   - `{{SAMPLING_DATE}}` - дата отбора
   - `{{PACKAGING}}` - вид упаковки
   - `{{PHYSICAL_STATE}}` - физическое состояние
   - `{{STATUS}}` - статус
   - `{{NOTE}}` - примечание

## Запуск

```bash
python app.py
```

## Запуск по клику (EXE)

1. В PowerShell в папке проекта выполните:

```powershell
.\build_exe.ps1
```

2. Запускайте файл:
`dist\RawMaterials\RawMaterials.exe`

3. Для переноса на другой ПК скопируйте папку `dist\RawMaterials\` целиком и при необходимости поправьте `settings.json`.

## Использование

1. **Новая запись**: Заполните форму и нажмите одну из кнопок статуса
2. **Редактирование**: Нажмите "Выбрать существующую запись" и выберите запись из списка
3. **Статусы**:
   - 🟢 **РАЗРЕШЕНО** - строка в Excel будет зеленой
   - 🟡 **КАРАНТИН** - строка будет желтой
   - 🔴 **БРАК** - строка будет красной
   - 🟠 **КОНТРОЛЬ** - строка будет оранжевой

## Структура проекта

- `config.py` - конфигурация путей
- `database.py` - работа с SQLite
- `documents.py` - работа с Excel и Word
- `app.py` - главное приложение с GUI
- `requirements.txt` - зависимости
